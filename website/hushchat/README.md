# HushChat Website

Marketing and product website for HushChat, built with [Astro](https://astro.build).

## Getting Started

```bash
npm install
npm run dev
```

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start dev server |
| `npm run build` | Build for production |
| `npm run preview` | Preview production build locally |

## Deployment

Deployed via GitHub Actions on `web-v*` tags (e.g., `web-v1.0.0`).
